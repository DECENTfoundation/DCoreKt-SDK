generatePrivateFromStringPhrase

@JvmOverloads fun generatePrivateFromStringPhrase(phrase: String, sequence: Int = 0, normalized: Boolean = true): ECKeyPair

Method generates private key from phrase provided by parameter of type String. If parameter normalize is true, provided pass phrase will be converted to upper case before private key calculation. In other case word stays as it was provided. Default value for normalize parameter is true.

Parameters

phrase - phrase string

sequence - private key derivation sequence

normalized - normalization flag
