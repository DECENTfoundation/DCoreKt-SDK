@JvmOverloads fun createFundPoolsOperation(assetIdOrSymbol: String, uia: Long, dct: Long, fee: Fee = Fee()): Single<AssetFundPoolsOperation>

Create fund asset pool operation. Any account can fund a pool.

Parameters

uia - UIA raw amount

dct - DCT raw amount

fee - Fee fee for the operation, by default the fee will be computed in DCT asset. When set to other then DCT, the request might fail if the asset is not convertible to DCT or conversion pool is not large enough
